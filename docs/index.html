Verify Service Provider API

POST /generate-request

Generate an authn request

Parameters

Parameter In Type Description
body body RequestGenerationBody No description (required)
» levelOfAssurance body string Level of assurance requested by the Service

Enumerated Values

Parameter Value
» levelOfAssurance LEVEL_1
» levelOfAssurance LEVEL_2

Example request body

{
  "levelOfAssurance": "LEVEL_1"
}

Responses

Status Meaning Description Schema
200 OK An object containing a SAML request. RequestResponseBody
422 Unprocessable Entity An error due to a JSON request in an invalid format (e.g. missing mandatory parameters). ErrorMessage
500 Internal Server Error An error due to an internal server error. ErrorMessage
This operation does not require authentication

Example responses

{
  "samlRequest": "string",
  "requestId": "string",
  "ssoLocation": "string"
}

{
  "code": 422,
  "message": "Some error message"
}

{
  "code": 500,
  "message": "Some error message"
}

POST /translate-response

Create a translated response

Parameters

Parameter In Type Description
body body TranslateSamlResponseBody An object containing a details of a SAML Authn response. (required)
» samlResponse body string(byte) A SAML Authn response as a base64 string. (required)
» requestId body string(byte) A token that was generated for the original SAML Authn request. The token is used to verify that the request and response are from the same browser. (required)
» levelOfAssurance body string Level of assurance the user authenticated with. (required)

Enumerated Values

Parameter Value
» levelOfAssurance LEVEL_1
» levelOfAssurance LEVEL_2

Example request body

{
  "samlResponse": "string",
  "requestId": "string",
  "levelOfAssurance": "LEVEL_1"
}

Responses

Status Meaning Description Schema
200 OK Response contains an object with details of a translated SAML response. TranslatedResponseBody
400 Bad Request an error due to a problem with translating the Response ErrorMessage
422 Unprocessable Entity An error due to a JSON request in an invalid format (e.g. missing mandatory parameters). ErrorMessage
500 Internal Server Error an error due to an internal server error ErrorMessage
This operation does not require authentication
